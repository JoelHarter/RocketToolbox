# RocketToolbox
航空宇宙の計算・解析のための標準的なJuliaユーティリティ集  

_2026年1月22日より開発中_

## 概要

### 📐 定数ライブラリ
* 単位換算
* 物理定数
* 地球および天体パラメータ  

### 🌏 座標系変換関数
* 地理座標系 (Geo)
* 地球中心地球固定座標系 (ECEF)
* 地球中心慣性座標系 (ECI)  

### 🔄 クォータニオンと回転
* クォータニオンオブジェクトの定義
* 2次元、3次元の主要軸、および一般回転行列のコンストラクタ

### 👽 その他、多彩な機能を搭載！

## セットアップ

### パッケージのインストール
Julia:
```
using Pkg
Pkg.add("StaticArrays")
```

### コード内での使用方法
Julia:
```julia
# include.jl が唯一のエントリポイントです。一度読み込むだけですべてが利用可能になります。
include("[リポジトリへのパス]/include.jl")
```

すべての関数、型、定数は現在のスコープに直接読み込まれます。
モジュール修飾は不要です：`Earth.μ`、`Quat(...)`、`rotmat(...)` などをそのまま使用できます。
